🩸 Blood Bank Management System

A full-stack Blood Bank Management System built using PHP, MySQL, HTML, CSS, and JavaScript.
This system helps manage donors, receivers, staff, and blood stock efficiently with a real-time dashboard.

---

🚀 Features

- 👨‍⚕️ Donor Management (Add, Update, View)
- 🏥 Receiver Management
- 🧑‍💼 Staff Management
- 🩸 Blood Stock Management
- 📊 Dashboard with Real-Time Analytics
- 🔍 Search and Filter (by blood group & name)
- 📈 Stock Distribution Graph
- ⚠️ Low & Critical Stock Alerts
- 🕒 Recent Activity Feed (Donors, Receivers, Staff)

---

🧱 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP

---

🗂️ Database Design

Main Tables:

- "donors"
- "receivers"
- "staff"
- "stock"

Key Concepts:

- Foreign Key Relationships (staff_id)
- Dynamic Stock Updates
- Activity Tracking using "created_at"

---

⚙️ Installation

1. Clone the repository:

git clone https://github.com/your-username/blood-bank-project.git

2. Move project to:

C:\xampp\htdocs\

3. Start:

- Apache
- MySQL

4. Import database:

- Open phpMyAdmin
- Create database: "blood_bank"
- Import ".sql" file

5. Run project:

http://localhost/Blood-Bank/
